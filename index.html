Killing Floor 2 Workshop Collection URL Converter

How to Use

  1. Type the URL from your maps (or items) collection of your Steam Workshop (example: 'put a collection example').
  2. Check if in your PCServer-KFEngine.ini (located on \KFGame\Config\) has a section named "[OnlineSubsystemSteamworks.KFWorkshopSteamworks]", if not, unmark the option "Exists [OnlineSubsystemSteamworks.KFWorkshopSteamworks] Section".
  3. Click Convert.
  4. Add the result to the end of the server PCServer-KFEngine.ini (located on \KFGame\Config\).

For more help, check this Article Section on Tripwire Wiki.
